\chapter{Enumerate Important Terms}
The relevant terms for both the base ontology and the new extensions are listed below:

\begin{itemize}
    \item \textbf{Nouns (candidate classes):}
    \begin{itemize}
        \item \textit{From Base:} Film, Actor, Director, Soundtrack, Artwork, Person, Composer, FilmStudio.
        \item \textit{New Extensions:} Series, Episode, Classification, Genre, AgeRating, Subscriber, StreamingPlatform, SubscriptionPlan.
    \end{itemize}
    \item \textbf{Verbs (candidate object properties):}
    \begin{itemize}
        \item \textit{From Base:} hasActor, hasDirector, hasCrew, prequel, sequel.
        \item \textit{New Extensions:} availableOn, watched, likes, subscribedTo, hasPlan, belongsToSeries, hasGenre, hasAgeRating.
    \end{itemize}
    \item \textbf{Descriptive nouns (candidate data properties):}
    \begin{itemize}
        \item \textit{From Base:} releaseYear, fullName, dateOfBirth, budget.
        \item \textit{New Extensions:} subscriberAge, subscriptionCost, imdbRating.
    \end{itemize}
\end{itemize}
