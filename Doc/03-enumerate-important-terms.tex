\chapter{Enumerate Important Terms}
The relevant terms for both the base ontology and the new extensions are listed below:

\begin{itemize}
    \item \textbf{Nouns (candidate classes):}
    \begin{itemize}
        \item \textit{From Base:} Film, Actor, Director, Soundtrack, Artwork, Person, Composer, FilmStudio, Performer, ScriptWriter, Producer, Cinematographer, Editor, CostumeDesigner, Genre, Trailer.
        \item \textit{New Extensions:} Series, Miniseries, Season, Episode, AgeRating, Subscriber, StreamingPlatform, SubscriptionPlan, BasicPlan, PremiumPlan, FamilyPlan, AdSupportedPlan, StudentPlan, Country, Review, Award, Oscar, Emmy, GoldenGlobe, PalmeDOr, SciFiFilm, EnsembleFilm, AwardWinningArtwork, AdultSubscriber, PositiveReview, AdFreePlan, FullAccessSubscriber.
    \end{itemize}
    \item \textbf{Verbs (candidate object properties):}
    \begin{itemize}
        \item \textit{From Base:} hasActor, hasDirector, hasCrew, hasComposer, hasScriptWriter, hasPerformer, hasFilmStudio, hasSoundtrack, hasGenre, prequel, sequel, friendOf, knows.
        \item \textit{New Extensions:} watched, likes, wroteReview, reviewsArtwork, subscribedTo, hasPlan, offersPlan, availableInCountry, availableOn, locatedIn, hasOriginCountry, hasAgeRating, hasAward, hasSeason, hasEpisode, nextEpisode, belongsToSeries, inSameUniverse, coStarredWith, recommendsTo, coWatched, inProgress.
    \end{itemize}
    \item \textbf{Descriptive nouns (candidate data properties):}
    \begin{itemize}
        \item \textit{From Base:} releaseYear, fullName, dateOfBirth, budget, duration, gender, establishedDate.
        \item \textit{New Extensions:} subscriberAge, subscriptionCost, maxSimultaneousStreams, hasAds, platformURL, reviewScore, imdbRating, episodeNumber, seasonNumber, synopsis, originalLanguage, registrationDate, awardYear.
    \end{itemize}
\end{itemize}
